# ATS Scoring & Intelligent Matching Engine

An advanced Applicant Tracking System (ATS) scoring engine that evaluates candidate resumes against target job descriptions and generates visual hiring readiness dashboards and downloadable PDF reports.

## Features
- Resume Parsing (PDF & DOCX)
- Job Description Analysis
- Local Semantic NLP Matching Engine
- Interactive Dashboard Analytics (SVG charts, Heatmaps, Score Gauges)
- Historical Scan Tracking & Comparisons
- Professional PDF Report Generation
